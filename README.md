## Contribute

An issue will be open per week, if you want to contribute open the issue and paste a resource you think it's worth sharing in the newsletter and comment on the issue of the week. 

If there's no issue for the week open one and add a comment.

Use comment reactions on issue comments to upvote/downvote/whatever each submition the reactions that have a clear sense of positive/negative will be considered to rank the sumissions of the week.
