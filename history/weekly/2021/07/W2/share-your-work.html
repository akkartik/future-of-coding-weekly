Jamie Brandon 2021-07-10 05:30:48

Dida updates:

  • Added a js<->wasm api (example here, sugar upcoming so it looks more like this)](https://github.com/jamii/dida/blob/2caffd4461037fe5c06b2457251a94e74cfda1dc/examples/sugar.zig))). From the js side the wasm api looks exactly the same as the node api, so you can share code between them.

  • Added a reduce operator.

  • Lots more unit tests.
