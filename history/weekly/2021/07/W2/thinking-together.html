Jack Rusher 2021-07-11 15:12:57

Kartik Agaram what I mean by semantics is that, for example:

  • Clojure's reduce takes a function, an optional initial value, and a sequence. (reduce #(assoc %1 (str %2) %2) {} [1 2 3 4]) ; => {"1" 1, "2" 2, "3" 3, "4" 4}
  • Ruby calls this collect, but instead of being a function it's an Array method that doesn't take an initial value: [ "a", "b", "c", "d" ].collect { |x| x + "!" } #=> ["a!", "b!", "c!", "d!"]
  • APL calls the same operation reduce, but spells it / and uses it as a modifier to another operation: +/1 2 3 4 5 ⍝ 15

... and so on. These are all translations of the same concept, but they have cultural differences that make an exact translation impossible.
