Emmanuel Oga 2021-01-21 07:29:30

relaxng (XML validator) has an https://relaxng.org/compact-tutorial-20030326.html#id2815185 operator for this sort of thing. Its been a while since I last wrote a gramamr but I think it would look something like this:

grammar {

  start = element house { room*}

  room = element room { box & ((red & blue?) | (blue & red?)) }

  box = elem box { "box" }

  red = elem ball { "red"}

  blue = elem ball { "blue" }

}
Emmanuel Oga 2021-01-21 07:30:14

or else any other grammar would help (ANTLR, PEG, etc):

grammar = "house" room*

room = "box" ("red ball" "blue ball"? | "blue ball" "red ball"?)
Emmanuel Oga 2021-01-21 07:30:40

I think it is useful to think in terms of grammars because then becomes obvious when the thing you are trying to validate is not context free
