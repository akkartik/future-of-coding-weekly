Chris Maughan 2022-11-11 10:22:22

My tests boil down to a tonne of macros like this:

The strings are:

'name', 'start text', 'commands to the editor', 'expected result'.

COMMAND_TEST(paste_p_linewise, "(one)\ntwo\n", "Vyjp", "(one)\ntwo\n(one)\n");

This one tests that you can select the whole line, yank it, move down a line, paste it below

I think a set of tests like this are essential, since as soon as you tweak the guts of the editor, it's really easy to break a small feature somewhere and not notice for a while.
